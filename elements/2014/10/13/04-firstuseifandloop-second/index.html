配列ってなーに

はい、配列っていうのは感ターンに言うと
あるTという型があったとしてT型の配列というと
T型の値がN個格納された変数です
宣言するときはT[] 変数名;と宣言します
わかりにくいかもしれないので実際にサンプルコードを書いてみます

これでarrayというint型の要素が5つの動的配列を宣言したことになります
動的配列とはどういうことかというと
処理中に自由に配列の長さを変えることが出来る配列ということです
つまり

こう書いた場合
最初は
[0, 1, 2, 3, 4]
ですが、
次は
[0, 1, 2, 3, 4, 5]
になっていますよね
確かめてみてください

解説にはいりますね
配列名.lengthと書くことで配列の長さにアクセスすることができます
で、配列の長さをインクリメントすることで配列の長さを1つ増やしています
次にarray[array.length - 1] = 5とかいて新しく用意したスペースに5を代入しています
ん？[]ってなに? なんで1引くの?ってなると思うので説明します

まず[]は添字を意味します
添字とは配列のどこを指すかということで
[1, 2, 3, 4]という整数型の配列Kがあった場合
K[2]は3を示します
あれ>? 2じゃないの?? ってなると思うんですよー
でも、違うんです 3なんです
理由は添字が0から始まるからです
つまり
配列 : [1, 2, 3, 4]
添字 : 0 1 2 3
と対応していますなので目で見た時の要素の場所 - 1が実際の場所になります
で、要素数は普通に1から始まるので帳尻を合わせるために1をlengthから引いているわけです

まとめると配列というのは
ある型の値の列ということですね
